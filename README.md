to run server on your localhost:
> open terminal<br>
> `cd bas-it-test/server`<br>
> `npm i`<br>
> `npm run start`<br>

to initiate database on your localhost
> open terminal<br>
> `cd bas-it-test/server`<br>
> `npm run up`<br>

to reset database on your localhost
> open terminal<br>
> `cd bas-it-test/server`<br>
> `npm run down`<br>
> `npm run up`<br>
